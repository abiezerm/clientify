## About The Project

This application allows the companies to manage customers and add to them multiple addresses.

### Built With

- [NextJs](https://nextjs.org/)
- [Firebase](https://firebase.google.com/)
- [React](https://reactjs.org/)


### Installation using Docker

1. Go to src folder and build the docker image ``` docker build --pull --rm -f "DockerFile" -t clientify:latest "." ```
2. Run a container using the created image ``` docker run --rm -d  -p 3000:3000/tcp clientify:latest ```
